1. The Code-Generation Bottleneck

Today, AI coding agents rely on slow, heavy Docker containers or Firecracker microVMs to test generated code. This limits them to testing 1-2 permutations per second. Trytet compiles generated code to fuel-bounded WebAssembly, increasing agent execution speed by 10,000x.

  • Sub-Millisecond Boot: Evaluate untrusted Python or JS instantly.
  • Infinite Loop Protection: Instruction-level fuel limits trap infinite loops automatically. Your testing pipeline never hangs.
  • Massive Concurrency: Test thousands of generated code variations in parallel on a single node.
src/agent.ts [TYPESCRIPT SDK]
// Wrapping an autonomous agent in Trytet
import { TrytetRuntime } from '@trytet/sdk';

const agent = new TrytetRuntime({
    wasmBinary: './agent-core.tet',
    maxFuel: 50000, // deterministic budget
});

// Run the agent dynamically
const execution = await agent.run({ 
    task: "Refactor this repo" 
});

// Forking the agent exactly at crash state 
if (execution.error) {
    const clone = await TrytetRuntime.fork(
        execution.snapshot_id
    );
    await clone.continue();
}

2. High-Frequency DeFi Agents

Trading agents lose money when transaction simulations are too slow or unstable. Trytet allows an agent to call a deterministic Cartridge replica of a liquidity pool to simulate a trade in microseconds.

  • Zero-Latency Simulations: Test complex arbitrage opportunities against local state without round-trip latency.
  • Cryptographic Certainty: Cartridges trap instantly on mathematical failures, allowing the agent to move to the next opportunity.
  • Resource Bounding: The host is guaranteed not to crash, keeping your high-frequency bot online 24/7.
src/evaluator.py [PYTHON SDK]
import trytet

# Define strict boundaries before execution
env = trytet.Environment(
    max_memory_mb=128,
    vfs_mounts={"/workspace": "./isolated"},
    network_egress=False
)

# LLM generated string
untrusted_code = "import os; os.system('rm -rf /')"

# Try to run it in the ephemeral sandbox
try:
    result = trytet.spawn(code=untrusted_code, env=env)
